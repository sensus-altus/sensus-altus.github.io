CSS display and positioning cheatsheet

Here are some key concepts relating to display and positioning in CSS, covered by Codecademy.


HTML tag Values Description
position static Default position for an element.
relative Positioned relative to element’s default position.
absolute Positioned relative to element's parent element. Other elements behave as if the element is not present. Allows for overlap.
fixed Positioned relative to the browser window. Element is removed from flow of document and stays in the same position as user scrolls. Always in view.
sticky Like fixed but allows reset of the elements to scroll over to cover it.
z-index integer values Depth of elements controlling order of layered elements.
Doesn’t work on static elements.
display inline Make any element inline.
block Make any element a block.
inline-block Displays blocks that can be side by side but with specified width and height.
float left Align left and wrap other elements (like text) around it. Must have width specified.
right Align right and wrap other elements around it.
clear left Left of element will not touch any other element within the same containing element.
right Right of element will not touch any other element within the same containing element.
both Neither side of element will touch.
none Element can touch either side (default).
